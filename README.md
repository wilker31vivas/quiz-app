# 📝 Quiz App

> A dynamic quiz application built with React + TypeScript. Create your profile, pick an avatar, and test your knowledge with questions pulled live from the Open Trivia Database API.

🌐 **[Live Demo →](https://quiz-wilker.netlify.app)**

## ✨ Features

- 👤 **User profile** — Create an account with a custom username and avatar
- 🖼️ **Avatar selection** — Choose from a variety of available avatars
- ❓ **Dynamic questions** — Real-time questions fetched from the [Open Trivia Database API](https://opentdb.com/api_config.php)
- 📊 **User statistics** — Track your score, streaks and performance history
- 🗑️ **Delete account** — Remove your profile and all associated data

## ⚙️ Installation

### Prerequisites

- Node.js >= 18.0.0
- npm

### Steps

**1. Clone the repository**

```bash
git clone https://github.com/tu-usuario/quiz-app.git
cd quiz-app
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

**4. Open in your browser**

```
http://localhost:5173
```

> 🌐 The app fetches questions directly from the Open Trivia Database — no API key required.
